Real-Time Project

E-ship

At E-ship Simplified, I worked on a logistics platform designed to optimize the shipping process for small to medium-sized businesses. My key contribution was improving the app's scalability by leveraging Redux for state management. For example, the platform needed to handle dynamic shipment data updates, and the previous implementation struggled under load. I refactored the state management to Redux, reducing API calls by caching frequently accessed data, which significantly improved performance. Additionally, I enhanced the user experience using React Router by enabling smooth navigation between different views, such as shipment tracking, invoicing, and customer support. This ensured seamless transitions and made the platform more intuitive, especially for users unfamiliar with tech-heavy interfaces.

Dreammill

At Dreammill, I was part of a team developing an e-learning platform for schools and individual learners. My role was focused on creating an interactive single-page application (SPA) using ReactJS. One of my major achievements was implementing responsive design for the course catalog and student progress dashboard. I worked closely with the UI/UX team to ensure the platform was accessible across various devices, from smartphones to desktops. I also implemented DOM manipulation techniques for real-time updates, such as progress tracking and live quizzes. This significantly increased user engagement by making the platform visually appealing and functionally seamless.
